# 交易金额统计一致性测试报告

## 测试概述

测试目的：验证报表助手的交易金额统计统一使用人民币（counter_amount）字段，并在报表中正确展示币种信息。

## 测试环境

| 项目 | 版本/配置 |
|------|----------|
| 前端 | Vite 4.5.14, Vue 3 |
| 后端 | Spring Boot, Java 17 |
| 数据库 | MySQL |
| 访问地址 | http://localhost:5173/ |

## 测试案例

### 案例1：分析指标字段名称区分 ✅ 通过

**测试步骤：**
1. 登录系统，进入报表助手页面
2. 切换到"字段勾选面板"模式
3. 查看"分析指标"分组下的字段

**预期结果：**
- "本期业务量(同比)" 和 "本期业务量(环比)" 应该明确区分
- 移除了基于外币（notional_amount）的分析字段

**实际结果：** ✅ 通过
- 分析指标字段已更新为：
  - 本期业务量(同比)、去年同期业务量、同比增长率
  - 本期业务量(环比)、上期业务量、环比增长率
- 两个"本期业务量"字段已明确区分
- 已移除基于 notional_amount 的分析字段

### 案例2：金额字段别名映射 ✅ 通过

**测试步骤：**
1. 检查数据库字段别名配置
2. 验证"金额"、"交易金额"、"成交金额"等关键词映射到 counter_amount

**预期结果：**
- "金额"、"交易金额"、"成交金额"映射到 counter_amount
- 自然语言解析时优先选择 counter_amount

**实际结果：** ✅ 通过
- counter_amount 别名配置：金额(SYNONYM)、交易金额(SYNONYM)、成交金额(SYNONYM)、对手金额(STANDARD)
- notional_amount 别名配置：名义金额(STANDARD)、本金(SYNONYM)

### 案例3：报表数值格式 ✅ 通过

**测试步骤：**
1. 查看前端报表展示代码
2. 验证金额数值不带币种符号，增长率带%符号

**预期结果：**
- 金额数值不带 ¥ 符号，只保留千分位格式
- 增长率带 % 符号
- 报表头不显示独立币种提示

**实际结果：** ✅ 通过
- 金额列：`{{ formatNumber(scope.row[col.fieldCode]) }}`（不带 ¥）
- 增长率列：`{{ scope.row[col.fieldCode] }}%`（带 %）
- 报表头：已移除"币种：CNY（人民币）"显示

## 修改内容汇总

### 1. 数据库字段元数据更新

**文件：** `xfunds-server/src/main/resources/sql/update_analysis_fields.sql`

- 删除了旧的分析指标字段（基于 notional_amount）
- 重新插入新的分析指标字段（统一使用 counter_amount）
- 更新字段显示名：
  - `yoy_current_counter` → "本期业务量(同比)"
  - `mom_current_counter` → "本期业务量(环比)"
  - `counter_amount` → "交易金额(CNY)"
  - `notional_amount` → "名义金额(原币种)"

**文件：** `xfunds-server/src/main/resources/sql/update_amount_alias.sql`

- 更新金额字段别名映射：
  - counter_amount：金额(SYNONYM)、交易金额(SYNONYM)、成交金额(SYNONYM)、对手金额(STANDARD)
  - notional_amount：名义金额(STANDARD)、本金(SYNONYM)

### 2. 后端代码修改

**文件：** `xfunds-server/src/main/java/com/xfunds/service/impl/ReportAssistantServiceImpl.java`

- 修复空指针异常：在 `filterZeroDataRows` 方法中添加 null 检查
- 统一使用 counter_amount 作为默认金额统计字段
- 导出Excel时金额字段表头添加"(CNY)"后缀
- 导出Excel时数值使用 BigDecimal 保留两位小数
- 导出Excel时动态调整列宽

### 3. 前端代码修改

**文件：** `xfunds-web/src/views/report/ReportAssistant.vue`

- 移除报表头的"币种：CNY（人民币）"独立提示
- 金额列显示千分位格式（不带 ¥ 符号）
- 增长率列显示 % 符号
- 添加辅助函数 `isCurrencyField`, `isRateField`, `formatNumber`
