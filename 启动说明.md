# 银行外汇交易系统 - 启动说明

## 系统概述
这是一个功能完整的银行外汇交易系统，包含前后端完整代码。

## 系统要求
- Java 17+
- Node.js 16+
- MySQL 8.0+
- Maven 3.6+

## 快速启动

### 1. 确认MySQL服务运行
确保MySQL服务在运行，用户名root，密码aa111111

### 2. 启动后端服务
```powershell
cd d:\trae\xfunds\xfunds-server
java -jar target\xfunds-server-1.0.0.jar
```
后端将在 http://127.0.0.1:8080 启动

### 3. 启动前端服务（新开一个终端
```powershell
cd d:\trae\xfunds\xfunds-web
npm run dev
```
前端将在 http://127.0.0.1:5173 启动
